# Documentação da Linguagem de Programação AuroraLogo v1.01

### Funções Matemáticas

#### valorAbsoluto

```
valorAbsoluto( valor )
```

Calcula e retorna o valor absoluto de um valor numérico.

**Parâmetro(s):**
 - **valor:** um valor numérico.

**Retorna:** O valor absoluto do valor passado como parâmetro.
